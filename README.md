# 游戏demo地址 https://easymapeditor-1258223435.cos.ap-guangzhou.myqcloud.com/frameworkdemo/demo3/web-desktop/index.html  
下载游戏项目代码后，进行以下操作：
1、下载框架插件

windows
执行根目录下的 update-cocos-framework.bat 下载最新版本框架插件

mac
项目下载后执行 update-cocos-framework.sh 下载最新版本框架插件
(mac需要可执行文件权限，在当前项目目录下执行 sudo chmod a+x ./update-cocos-framework.sh)

2、下载好框架代码后需要重启creator


打开地图资源：
1、先克隆地图资源仓库 https://github.com/cyk88888888/cocos-mapDataRes.git
2、在地图编辑器左上角点击打开按钮，选择地图资源的其中一个文件夹即可打开
